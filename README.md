# VYBZ Landing Page

Official landing page for VYBZ - Real Vibes Only.

## Tech Stack

- Static HTML
- Tailwind CSS (CDN)
- GSAP Animations
- Spline 3D Background

## Development

Open `index.html` in a browser. No build step required.

## Deployment

Deployed automatically to Vercel on push to `main`.

## Analytics Setup

Analytics placeholders are included in `index.html`. To enable:

1. **Google Analytics 4**: Replace `G-XXXXXXXXXX` with your GA4 measurement ID
2. **Facebook Pixel**: Replace `XXXXXXXXXXXXXXXXX` with your Pixel ID

Then uncomment the relevant script blocks.
